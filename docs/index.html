About

b-oss = Blipfoto - Open Source Software
Open source means anyone can see the source code, build the software themselves, modify it, and contribute improvements back. If you have the skill and the interest, you can join the development team.

b-oss was started by Ian Stevenson — a technologist with thirty years' experience as a software engineer and running software companies — using Claude Code and other AI tools for implementation. Ian is also one of the directors of blipfoto, but b-oss is a personal project and has no formal relationship with blipfoto.

b-ark

Archive your blips on your own device — for Windows.

b-ark keeps a local copy of every blip you've ever posted — images, text and metadata — on your own machine. Browse them offline, publish them straight to the web, or hand them to AI tools to do something new with your archive.

  • Back up. A local copy of every image, every word, every blip.
  • View. Browse your archive offline in the built-in viewer.
  • Publish. Drop the backup folder on a web host — that's your site.
  • AI-ready. Each backup ships with a README so AI tools can read your data — build photobooks, slideshows, statistics, custom sites.
